## Set-up

> On host

1. `git clone `
1. `vagrant up`

> On cuckoo host

1. `git clone https://github.com/terib0l/cuckoo-vm.git`
1. `sh install.sh`
1. `cuckoo`
1. `cuckoo web`
1. goto localhost:8000
